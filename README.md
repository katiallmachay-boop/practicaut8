## Descripción
Aplicación web para gestionar las inscripciones y talleres de un campamento.

## Requisitos
* Java 17
* MySQL 8.0
* Spring Boot 3.x

## Pasos para ejecutar
1. Ejecutar el script `schema.sql` en MySQL.
2. Configurar `application.properties` con tus credenciales.
3. Ejecutar: `mvn spring-boot:run`
